## 地址选择组件

省市级三级联动选择
```html example
<city-area 
      v-model="cityAddr"
      placeholder="请选择"
    >
</city-area>
```

## Attributes

### value

说明：`v-model双向绑定值`
<br/>
类型：`String`
<br/>
默认值：`null`

### placeholder

说明：`placeholder`
<br/>
类型：`String`
<br/>
默认值：`null`

### addrstyl

说明：`可传入的class样式`
<br/>
类型：`String`
<br/>
默认值：`null`
